Remote repository

To pull remote repo use:

git remote add origin https://github.com/x/y.git

git pull origin

or

git remote add origin https://github.com/x/y.git

git fetch origin

git merge origin/master

To make remote repository use local:

git init --bare

git remote add origin URL

git push origin master
